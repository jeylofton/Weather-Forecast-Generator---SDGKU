# Weather-Forecast-Generator---SDGKU

Lessons Learned

Working on this weather forecast project taught me several valuable lessons about web development and project management.

Save Stable Versions of Your Work

One of the biggest lessons I learned was the importance of saving stable versions of my code. As I made changes throughout the project, there were times when a new change caused unexpected problems. Having previous working versions available made it much easier to recover and continue moving forward. Going forward, I plan to save and commit my work whenever I reach a working milestone rather than waiting until the entire project is complete.

The Importance of Using Div Containers

Another lesson I learned is how important <div> elements are when building a webpage. In the past, I often placed elements directly on the page without grouping them together. During this project, I learned that wrapping related elements inside a <div> allows me to style and manipulate them as a single unit using CSS and JavaScript. This became especially important when creating the forecast cards and making the layout responsive for mobile devices.

Understanding the Relationship Between HTML, CSS, and JavaScript

This project also helped me better understand how HTML, CSS, and JavaScript work together. One concept that stood out to me was learning that JavaScript can generate HTML dynamically and insert it into the page using innerHTML. This allowed me to create forecast cards using JavaScript while still controlling their appearance with CSS.

Responsive Design Matters

I learned that designing for desktop is only part of the process. A website should also work well on tablets and mobile devices. Through the use of media queries, I was able to create a layout that displays forecast cards in a row on larger screens while switching to a more mobile-friendly stacked layout on smaller screens.

Problem Solving and Debugging

Perhaps the most important lesson was learning how to troubleshoot problems. Throughout the project, I encountered issues with CSS layouts, JavaScript output, and responsive design. By testing changes, reading error messages, and making small adjustments, I was able to solve problems and better understand how the code was working behind the scenes.

Final Reflection

This project reinforced the idea that web development is a process of continuous learning. While I am becoming more comfortable with HTML and CSS, I still have more to learn with JavaScript. Each project helps me build confidence and gives me a better understanding of how websites are structured, styled, and made interactive.
